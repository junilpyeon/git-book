# 사원별 기술 등급 관리

**사원별 기술 등급 관리**

* 부서별로 사원리스트를 확인합니다.
* 수정버튼을 클릭하여 해당 사원의 등급과 변경 적용 일자를 지정하여 체크후 저장하기 버튼을 누릅니다.
* 최근 적용 일자에 마우스오버하여 등급 변경이력을 최근 3건까지 확인합니다.
