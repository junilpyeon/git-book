# 작업 배정 관리

<figure><img src="../.gitbook/assets/4 (2).png" alt=""><figcaption></figcaption></figure>

### **작업 배정 리스트 항목**&#xD;

* 배정 직접  등록 : 배정 권한자(본부장,  팀장)가 직접 영업관리번호를 선택하여 엔지니어 배정을 작성합니다.
* 배정 리스트 : 항목을 선택하여 들어온 지원 요청 상세 화면을 확인합니다.
